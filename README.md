# PHP HttpQuery

## Installation

Via Composer

### Adding dependences to composer
```json
    {
        "minimum-stability": "dev",
        "repositories": [
            {
                "type":"git",
                "url":"https://github.com/TsSaltan/php-HttpQuery"
            }
        ],
        "require": {
            "tssaltan/php-HttpQuery": "dev-master"
        }
    }
```
``` bash
$ composer require https://github.com/TsSaltan/php-HttpQuery
```
